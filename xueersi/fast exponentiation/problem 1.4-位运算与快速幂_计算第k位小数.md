# 题目描述

给出正整数 a, b，计算分数 a/b 化成十进制小数后，小数点后第 k 位的数码。

## 输入格式

输入包含多组数据：
- 第 1 行，1 个正整数 T 表示有 T 组测试数据
- 接下来 T 行，每行 3 个整数 a, b, k

## 输出格式

输出 T 行，对于每组数据，输出 a/b 小数点后第 k 位

## 输入样例 #1

```
4
1 7 2
1 65536 5
7 91 200
1 998244353 998244352
```

## 输出样例 #1

```
4
1
7
3
```

## 数据说明

- 1 ≤ T ≤ 1000
- 1 ≤ a, b ≤ 10^9
- 1 ≤ k ≤ 10^9
