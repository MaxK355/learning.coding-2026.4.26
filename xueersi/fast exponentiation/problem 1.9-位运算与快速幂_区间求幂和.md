# 题目描述

小猴有一个 $n$ 长数列 $a^{b_1}, ..., a^{b_n}$，现在他有 $q$ 个询问：区间 $[l,r]$ 中项的乘积是多少？即：

$$a^{b_l} \times ... \times a^{b_r}$$

举个例子，假设 $a=2$，数列为 $2^3, 2^7, 2^1, 2^0, 2^2$，那么区间 $[2,4]$ 中项的乘积为：

$$2^7 \times 2^1 \times 2^0 = 256$$

请你依次计算小猴的询问。

## 输入描述

第一行三个正整数 $n,a,q$。

第二行 $n$ 个非负整数 $b_1, ..., b_n$。

接下来 $q$ 行，每行两个正整数 $l,r$。

## 输出描述

$q$ 行，每行一个整数表示结果，结果可能过大，请对 $10^9+7$ 取模。

## 样例1

**输入**
```
5 2 3
3 7 1 0 2
2 4
1 2
3 5
```

**输出**
```
256
1024
8
```

## 样例2

**输入**
```
3 3 2
1 3 2
1 2
2 3
```

**输出**
```
81
243
```

## 提示

对于 $30\%$ 的数据：$1 \le n,q \le 10^2$，$1 \le a \le 10^2$，$0 \le b_i \le 10^2$。

对于 $100\%$ 的数据：$1 \le n,q \le 10^5$，$1 \le a \le 10^5$，$0 \le b_i \le 10^9$。
