# 题目描述

有 $n$ 个人围成一圈，编号从 $1$ 到 $n$（$n$ 号挨着 $1$ 号）。

现在从 $1$ 号开始报数，第一回合报到 $m^{k+1}$ 的人就出列，第二回合从出局的下一个人开始报数，报到 $m^{k+2}$ 的人出列，……，第 $i$ 回合从第 $i-1$ 回合出局的下一个人开始报数，报到 $m^{k+i}$ 的人出列，依次类推，直到剩下一个人，请输出最后一个人的编号。

## 输入描述

一行两个整数 $n,m,k$。

## 输出描述

一行一个整数，表示最后剩下的人的编号。

## 样例1

**输入**
```
5 2 0
```

**输出**
```
5
```

## 样例2

**输入**
```
10 100 20
```

**输出**
```
8
```

## 样例3

**输入**
```
100 100 1000000
```

**输出**
```
43
```

## 提示

对于 $40\%$ 的数据：$1 \le n \le 100$，$1 \le m \le 10$，$0 \le k \le n$；

对于 $100\%$ 的数据：$1 \le n \le 10^4$，$1 \le m \le 10^9$，$0 \le k \le 10^9$。
