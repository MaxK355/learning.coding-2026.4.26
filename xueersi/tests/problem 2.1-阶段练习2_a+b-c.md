# 题目描述

计算：a+b-c

## 输入描述

三个整数 a,b,c

## 输出描述

结果

## 样例1

**输入**
```
1 1 1
```

**输出**
```
1
```

## 样例2

**输入**
```
1 2 4
```

**输出**
```
-1
```

## 样例3

**输入**
```
8888888888888888888888888888888
1111111111111111111111111111111
4
```

**输出**
```
9999999999999999999999999999995
```

## 提示

$1 \le a,b,c \le 10^{100}$
