# 题目描述

定义多项式 $h_n(x)$ 如下：

$$h_n(x) = \begin{cases} 1, & n=0 \\ 2x, & n=1 \\ 2xh_{n-1}(x) - 2(n-1)h_{n-2}(x), & n>1 \end{cases}$$

对给定的 $x$ 和正整数 $n$，求多项式 $h_n(x)$ 的值。

## 输入描述

输入 $n$ 和 $x$。

## 输出描述

求多项式的值。

## 样例1

**输入**
```
2 4
```

**输出**
```
62
```

## 提示

**样例#1解释**

对于 $h_2(4)$，我们使用第三个条件，因为 $n=2>1$：

$$h_2(4) = 2*4*h_1(4) - 2(2-1)*h_0(4)$$

现在我们需要计算 $h_1(4)$ 和 $h_0(4)$：

- $h_1(4) = 2x = 2*4 = 8$（使用 $n=1$ 的条件）
- $h_0(4) = 1$（使用 $n=0$ 的条件）

将这些值代入步骤 1 的公式：

$$h_2(4) = 2*4*8 - 2(1)*1 = 64 - 2 = 62$$

因此，$h_2(4)$ 的值是 $62$。

$n \le 8，x \le 8$
