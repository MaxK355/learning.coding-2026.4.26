# 题目描述

一个由 N 个整数构成的集合 {a1, a2, ..., aN}。在这里，我们定义集合的"大小"为集合中最大元素和集合中最小元素的差。例如，集合 {1,7,4} 的最大值7最小值1，所以这个集合的大小为 7-1=6。

现在想要把这个 N 个数的集合划分成 K 个集合，使得 K 个集合的"大小"的总和最小。

## 输入描述

第1行，2个正整数 N, K

第2行，N 个整数 {a1, a2, ..., aN}

## 输出描述

K 个集合的"大小"的最小总和

## 样例1

**输入**

```
5 2
1 10 11 2 9
```

**输出**

```
3
```

## 样例2

**输入**

```
4 2
0 7 3 13
```

**输出**

```
7
```

## 样例3

**输入**

```
3 3
1 10 100
```

**输出**

```
0
```

## 样例4

**输入**

```
10 3
1 12 19 23 28 3 32 36 41 7
```

**输出**

```
28
```

## 提示

样例1：划分成集合 {11,9,10} 和 {2,1}。{2,1} 的大小为1，{11,9,10} 的大小为2，总和为3。

样例2：划分成集合 {0,7,3} 和 {13}。

样例3：划分成集合 {1}、{10}、{100}

样例4：划分成集合 {12}、{28,23,41,19,32,36}、{3,1,7}

数据范围：1≤K≤N≤5×10^4，0≤ai≤10^12
