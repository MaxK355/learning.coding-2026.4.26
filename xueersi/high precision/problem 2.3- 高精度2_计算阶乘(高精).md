# 高精度2_计算阶乘(高精)

## 题目描述

输入整数 $n$，计算 $n$ 的阶乘 $n!$。

$$n! = 1 \times 2 \times 3 \times \cdots \times n$$

规定 $0$ 的阶乘等于 $1$。

## 输入格式

一个整数 $n$。

## 输出格式

输出 $n$ 的阶乘。

## 样例

### 样例 #1

**输入：**
```
50
```

**输出：**
```
30414093201713378043612608166064768844377641568960512000000000000
```

## 数据范围

- $0 \le n \le 200$
- 保证输出不超过 $1000$ 位
