# 高精度1_小猴的数字

## 题目描述

小猴发明了一种表示数字的方法: 给出一个开头数 $L$ 和一个结尾数 $R$ ($L<R$)，小猴将 $L$ 到 $R$ 的十进制数从左到右写在一起得到的数字记为 $(L\sim R)$

例如:
- $(4\sim 8)$ 表示 $45678$
- $(8\sim 12)$ 表示 $89101112$
- $(98\sim 110)$ 表示 $9899100101102103104105106107108109110$

小猴现在有两个这样表示的正整数, 他想计算两个正整数的和

## 输入描述

输入有2行, 每行2个正整数:

- 第1行2个正整数 $L_1, R_1$, 以空格分隔
- 第2行2个正整数 $L_2, R_2$, 以空格分隔

表示小猴想要计算 $(L_1\sim R_1)+(L_2\sim R_2)$

## 输出描述

输出一行, 两个正整数之和的十进制表示

## 样例1

**输入:**
```
2 14
22 25
```

**输出:**
```
234567891033353739
```

## 样例2

**输入:**
```
35 52
82 99
```

**输出:**
```
1182022242628303234363840424446485051
```

## 提示

- $1 \le L_1 < R_1 \le 999$
- $1 \le L_2 < R_2 \le 999$
