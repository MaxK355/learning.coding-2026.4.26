# 高精度1_a+b(高精度)

## 题目描述

输入两个正整数 $a$ 和 $b$, 计算 $a+b$ 的值并输出

## 输入描述

两行, 每行一个正整数, 分别代表 $a$ 和 $b$

## 输出描述

一行, 输出 $a+b$ 的结果

## 样例1

**输入:**
```
999999999999999999999999
1
```

**输出:**
```
1000000000000000000000000
```

## 样例2

**输入:**
```
14561268494113694459841256
98745632145555569812221111
```

**输出:**
```
113306900639669264272062367
```

## 提示

$1 \le a, b \le 10^{1000}$
