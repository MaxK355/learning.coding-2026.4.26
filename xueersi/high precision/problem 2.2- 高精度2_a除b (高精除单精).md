# 高精度2_a除b (高精除单精)

## 题目描述

输入两个正整数 $a$ 和 $b$，$a$ 是 $b$ 的倍数，计算 $a \div b$ 的值并输出。

## 输入格式

两行，每行一个正整数，分别代表 $a$ 和 $b$。

## 输出格式

一行，输出 $a \div b$ 的结果。

## 样例

### 样例 #1

**输入：**
```
999999999999999999999999
9
```

**输出：**
```
111111111111111111111111
```

### 样例 #2

**输入：**
```
1634254846899862270311363684648
112233
```

**输出：**
```
14561268494113694459841256
```

## 数据范围

- $1 \le a \le 10^{1000}$
- $1 \le b \le 10^9$
