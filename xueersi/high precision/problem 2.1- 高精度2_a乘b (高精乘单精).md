# 高精度2_a乘b (高精乘单精)

## 题目描述

输入两个正整数 $a$ 和 $b$，计算 $a \times b$ 的值并输出。

## 输入格式

两行，每行一个正整数，分别代表 $a$ 和 $b$。

## 输出格式

一行，输出 $a \times b$ 的结果。

## 样例

### 样例 #1

**输入：**
```
999999999999999999999999
1
```

**输出：**
```
999999999999999999999999
```

### 样例 #2

**输入：**
```
14561268494113694459841256
112233
```

**输出：**
```
1634254846899862270311363684648
```

## 数据范围

- $1 \le a \le 10^{1000}$
- $1 \le b \le 10^9$
