# 高精度1_斐波那契(高精)

## 题目描述

斐波那契数列 $F_1, F_2, \cdots, F_k, \cdots$ 如下定义:

$$F_1 = F_2 = 1$$

$$F_k = F_{k-1} + F_{k-2}, \quad (k \ge 3)$$

给出正整数 $n$, 计算 $F_n$ 的值

## 输入描述

一行, 1 个正整数 $n$

## 输出描述

一行, 输出 $F_n$ 的值

## 样例1

**输入:**
```
10
```

**输出:**
```
55
```

## 样例2

**输入:**
```
99
```

**输出:**
```
218922995834555169026
```

## 提示

$1 \le n \le 4000$

$F_{4000}$ 的值不超过 1000 位
