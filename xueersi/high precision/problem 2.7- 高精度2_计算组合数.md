# 高精度2_计算组合数

## 题目描述

组合数 $C_n^m$ 定义如下：

$$C_n^m = \frac{n!}{m!(n-m)!}$$

其中 $n!$ 表示 $n$ 的阶乘，规定 $0$ 的阶乘等于 $1$。

输入正整数 $n, m$，计算组合数 $C_n^m$。

## 输入格式

一行，两个正整数 $n, m$，以空格分隔。

## 输出格式

一行，输出组合数 $C_n^m$ 的值。

## 样例

### 样例 #1

**输入：**
```
5 3
```

**输出：**
```
10
```

### 样例 #2

**输入：**
```
99 37
```

**输出：**
```
2154618614921181030658724688
```

## 数据范围

- $0 \le m \le n \le 200$
- 保证输出不超过 $1000$ 位
