# 题目描述

从左往右，如果每一位数字都小于等于其左边的数字，这样的数被称为下降数（仅由 1～9 组成），比如 864431。

给出 n 和 k，输出所有 n 位的下降数中从大到小排列后的第 k 个数。

## 输入描述

输入共 1 行：2 个正整数 n, k。

## 输出描述

输出共 1 行：输出从大到小第 k 个 n 位下降数。

## 样例 1

**输入**

```
3 30
```

**输出**

```
961
```

## 样例 2

**输入**

```
28 152156
```

**输出**

```
9999999999999998876542222111
```

## 提示

- 3 ≤ n ≤ 30
- 1 ≤ k ≤ 10^7
- 保证 k 不超过 n 位下降数的个数
