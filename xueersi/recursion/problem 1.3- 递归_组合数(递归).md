# 题目描述

从n个物品里面选出m个物品的方案数，记为C(n,m)。

对于递归过程如下理解：假设n个物品里面有一个物品最特殊，挑选时m个物品时，这个特殊物品假定不需要挑选出来，此时要从n个物品里选出m个物品，就相当于要从n-1个物品里面挑选出m个物品记为：C(n-1,m)。如果挑选出来物品中有这个特殊的物品，则挑选的方式就是从n-1个物品中挑选出m-1个物品记为：C(n-1,m-1)。

所以C(n,m)=C(n-1,m)+C(n-1,m-1)。

特别的当n和m相等时或者挑选的数是0时，方案数规定为只有1种，表述为C(n,n) = C(n,0) =1。

现在请你从电脑中输入两个正整数，分别表示n和m，求从n个物品中选择m个物品，总共有多少选择方法？

写一个递归函数 `int cmb(int n, int m)`, 递归求组合数C(n,m)。

## 输入描述

两个整数 n, m

## 输出描述

一个整数, 组合数 C(n,m) 的值

## 样例1

**输入**

```
5 3
```

**输出**

```
10
```

## 提示

0 ≤ m ≤ n ≤ 32
