# 题目描述

从左往右，如果每一位数字都大于等于其左边的数字，这样的数被称为上升数，比如 134468。

给出 n 和 k，输出所有 n 位的上升数中从小到大排列后的第 k 个数。

## 输入格式

2 个正整数 n, k

## 输出格式

输出从小到大第 k 个 n 位上升数。

## 输入样例 #1

```
3 30
```

## 输出样例 #1

```
149
```

## 输入样例 #2

```
28 152156
```

## 输出样例 #2

```
1111111111111112234568888999
```

## 数据说明

- 3 ≤ n ≤ 30
- 1 ≤ k ≤ 10^7
- 保证 k 不超过 n 位上升数的个数
