# 题目描述

给定一个范围 $N$ ($N \le 10^7$)，你需要处理 $M$ ($M \le 5 \times 10^5$) 个某数字是否为质数的询问（每个数字均在范围 $1 \sim N$ 内）

## 输入描述

第一行包含两个正整数 $N$、$M$，分别表示查询的范围和查询的个数。

接下来 $M$ 行每行包含一个不小于 $1$ 且不大于 $N$ 的整数，即询问该数是否为质数。

## 输出描述

输出包含 $M$ 行，每行为 Yes 或 No，即依次为每一个询问的结果。

## 样例1

**输入**

```
100 5
2
3
4
91
97
```

**输出**

```
Yes
Yes
No
No
Yes
```
