# 题目描述

皮皮请你求出区间 $[L, R]$ 上的所有质数。之后输出小于等于 $k$ 的在 $[L, R]$ 区间内的最大质数，如果不存在则输出 $-1$。

## 输入描述

输入共 2 行：

第 1 行，两个整数 $L, R$；

第 2 行，一个整数 $k$，查询的数在 int 范围内。

## 输出描述

输出共 2 行：

第 1 行，若干个空格隔开的整数，表示区间 $[L, R]$ 上的所有质数（升序输出）；

第 2 行，如果存在小于等于 $k$ 且在 $[L, R]$ 区间内的质数，则输出这个质数；否则输出 $-1$。

## 样例1

**输入**
```
3 10
7
```

**输出**
```
3 5 7
7
```

## 样例2

**输入**
```
5 20
4
```

**输出**
```
5 7 11 13 17 19
-1
```

## 提示

$1 \le L \le R \le 10000000$；$1 \le k \le 10000000$。
