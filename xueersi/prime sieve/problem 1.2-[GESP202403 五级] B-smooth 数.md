# [GESP202403 五级] B-smooth 数

## 题目描述

小杨同学想寻找一种名为 **B-smooth 数**的正整数。

如果一个正整数的最大质因子不超过 $B$，则该正整数为 $B$-smooth 数。

小杨同学想知道，对于给定的 $n$ 和 $B$，有多少个不超过 $n$ 的 $B$-smooth 数。

## 输入描述

第一行包含两个正整数 $n$ 和 $B$，含义如题面所示。

## 输出描述

输出一个非负整数，表示不超过 $n$ 的 $B$-smooth 数的数量。

## 样例

**输入**
```
10 3
```

**输出**
```
7
```

## 提示

### 数据规模与约定

| 子任务 | 得分 | $n$ | $B$ |
|--------|------|-----|-----|
| 1 | 30 | $n \le B \le 10^3$ | |
| 2 | 30 | $n \le 10^6$ | $B \le 10^3$ |
| 3 | 40 | $n \le B \le 10^6$ | |

对全部的测试数据，保证 $1 \le n, B \le 10^6$。
